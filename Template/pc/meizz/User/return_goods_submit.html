售后服务条款

1.终身免费清洗保养:

我们为您的首饰提供终身免费清洗保养的服务，让您的首饰能够焕然一新；

2.成本价改款:

如果您佩戴了一段时间后想重新换款式，我们可以提供您成本价改款的服务，让您的首饰永远走在流行的前沿；可抵值金额计算：

·原产品抵值=原产品实际购买价格（不收取折旧费）

·旧托回收金额=旧托实际可回收重量*回收当日美之钻回收价

·应补差价=新成品的工费+新托金费-旧托回收金额

3.免费修改手寸:

我们还提供戒指免费修改手寸服务（正负2号内，部分款式不宜于改款的，不建议修改，具体按款式而定）；

4.免费刻字：

我们为您的首饰提供免费个性刻字服务，让首饰打上专属于您的特殊烙印，彰显与众不同的贵宾特质；

5.免费维修、翻新、和质检：

也为您的首饰提供终生免费维修、翻新、和质检服务；

6.钻饰原价以小换大服务：

置换规则：

·以小换大的产品为美之钻珠宝销售的产品

·适合主钻单颗10分以上钻石产品

·持有美之钻珠宝销售单据和证书（证书无污损和折皱）

·一对一换购钻石产品，且换购价格至少高于原产品30%的正价钻石产品

·以小换大只限换购同类型证书

·钻饰以小换大的服务是原价补差价，是为会员提供的增值服务，故换购的钻饰产品不享受同期店内其它优惠活动

·此项业务仅限在体验中心实施，回收之前需先作鉴定检验。

可抵值金额计算：

·原产品抵值=原产品实际购买价格（不收取折旧费）

·应补差价=新产品价格—原产品抵值

7.退换货服务条款：

·当您在美之钻购买的产品出现质量问题，可以在收到货品15天内（以签收销售单据上的日期为准）退换购买的货品。我们将依据《中华人民共和国消费者权益保护法》、《中华人民共和国产品质量法》及其他相关法规，为您提供周到的退换货服务，美之钻15天退货承诺为您无忧购物提供保障和信心。美之钻所有承诺工作日，遇国家法定节假日相应顺延，敬请谅解。

·产品退换货注意事项：

需将原产品、赠品及所有原包装、质量鉴定书以及销售单据一并交由珠宝顾问。
